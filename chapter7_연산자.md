## 삼항 조건 연산자

자바스크립트의 유일한 삼항 연산자.

<aside>
👩🏻‍💻 조건식 **?** 조건식이 true일 때 반환할 값 **:** 조건식이 false일 때 반환할 값

</aside>

```jsx
var x = 2;

// 2 % 2는 0이고 0은 false로 암묵적 타입 변환된다.
var result = x % 2 ? "홀수" : "짝수";

console.log(result); // 짝수
```
